\subsubsection{Descrição Detalhada}
Cada pattern dentro do urlpatterns deve ter o formato\-: \begin{DoxyVerb}url(r'REGEX', VIEW),
\end{DoxyVerb}


onde R\-E\-G\-E\-X é a expressão regular que será utilizada para identificar a U\-R\-L, e V\-I\-E\-W o método correspondente que será chamado para processar a requisição.

É importante ressaltar que o método view deverá retornar obrigatoriamente um objeto do tipo Http\-Response.

Dentro do projeto, todas as views são chamadas de dentro da Factory. Para mais informações, leia a documentação do \hyperlink{MainUnit_8py}{Main\-Unit.\-py}.
